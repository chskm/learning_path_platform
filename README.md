# Learning Path Platform

A platform for personalized learning paths using FastAPI, Streamlit, AWS Lambda, and Aurora MySQL.

## Structure
- `backend/`: FastAPI backend
- `frontend/`: Streamlit frontend
- `ml_api/`: Lambda ML API
- `terraform/`: Infrastructure as Code

## Setup
1. Clone the repository.
2. Configure AWS credentials.
3. Run `terraform apply` in `terraform/`.
4. Deploy backend/frontend to Elastic Beanstalk.
5. Deploy ML API to Lambda.

## Requirements
- Python 3.11
- Terraform
- AWS CLI
- Git
